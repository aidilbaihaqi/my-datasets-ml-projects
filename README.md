# My ML Projects 🚀

A collection of my AI, ML, DL, NLP, LLM, and DS projects, which draw inspiration from various sources such as Kaggle, HuggingFace, the UCI Repository, and others.

## 📁 Repository Structure

- **`dataset/`**: Contains datasets used across various projects.
- **`[my-projects]/`**: Folder of the projects

## 🧠 Domains Covered

- **Artificial Intelligence (AI)**
- **Machine Learning (ML)**
- **Deep Learning (DL)**
- **Natural Language Processing (NLP)**
- **Large Language Models (LLM)**
- **Data Science (DS)**

## 🌟 Sources of Inspiration

The projects and datasets in this repository are inspired by and sourced from:
- [Kaggle](https://www.kaggle.com/)
- [HuggingFace](https://huggingface.co/)
- [UCI Machine Learning Repository](https://archive.ics.uci.edu/)
- And other open-source open data platforms.
